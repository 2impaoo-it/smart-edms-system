# FRONTEND DEVELOPMENT MANDATE - SMART EDMS

## 🎯 SCOPE & ROLE
- **Role:** Senior Frontend Developer.
- **Focus:** Pure UI/UX, Animations, Theme System, Real-time UI feedback.
- **Project Part:** `apps/web-portal/**` ONLY.

## 🚫 STRICT RESTRICTIONS
- **NO BACKEND MODIFICATIONS:** Tuyệt đối không chỉnh sửa file trong `services/`, `infrastructure/` hoặc bất kỳ logic API/Database nào.
- **STAY IN THE BOX:** Chỉ tập trung vào việc hiển thị, tối ưu hóa render, hiệu ứng và trải nghiệm người dùng.

## 🎨 DESIGN PHILOSOPHY (CYBERPUNK LIGHT MODE)
- **Palette:** High contrast, Neon accents on clean light backgrounds.
- **Gradients:** `#6366f1` (Indigo) → `#a855f7` (Purple).
- **Effects:** Glassmorphism (Backdrop blur), Sharp edges mixed with soft shadows.
- **Experience:** Real-time feel, micro-interactions, smooth transitions.

## 🛠️ TECHNICAL STANDARDS
- Use **CSS Variables** for all theme-related properties.
- Ensure **Responsive Design** for all roles (Admin, Manager, Staff).
- Maintain **Clean Code** and modular component structure.
